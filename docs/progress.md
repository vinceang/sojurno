## Where we are

**`sojurno-v2` is the trunk** ‚Üí GitHub `vinceang/sojurno` (public), deployed on Vercel at **`sojurno.com`** (custom domain; the `*.vercel.app` URL still resolves), with **Storybook hosted at `sojurno.com/storybook/`** in the same deployment.
